## **Movie Advisor App**
This is the repository for code of Movie Advisor App by Serhii Ivashchuk.

## **Technology stack**

* Java 8 (1.8.0_241)
* Spring Boot 2.1.8
* Maven
* Thymeleaf

## **Building**

To compile source code, build and package compiled code in .jar:
```
mvn clean package 
```

## **Running**

To start app via Maven:
```
mvn spring-boot:run
```

## **Running the tests**

To run tests via Maven:
```
mvn clean test
```
